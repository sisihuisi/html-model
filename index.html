	 如果你想写 
	if (!false) 
	{ 
	   alert(123123); 
	} 
	不妨考虑写成: 
	false || alert('false'); 
	false || alert('false'); true || alert('true'); //output false; 
	用"||"的情况下,第一个条件true,不检测第二个直接返回true.第一个条件false，会执行第二个条件检测 
	
	
	 如果你想写 
	if (true) 
	{ 
	   alert(123123); 
	} 
	不妨考虑写成: 
	true && alert(123123); 
	false && alert(123123); true && alert(123123); //output true 
	用"&&"的情况下,第一个条件true，还会检测第二个条件。第一个条件false，直接返回false退出。
